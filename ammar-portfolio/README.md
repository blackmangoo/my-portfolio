# Ammar Akbar — AI Engineer Portfolio

Modern portfolio built with Next.js, Tailwind CSS, and Framer Motion, showcasing AI engineering experience, featured projects, and technical stack.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- lucide-react icons

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Project Structure

- `src/app/page.tsx` — main portfolio page and section data
- `src/app/layout.tsx` — root layout and metadata
- `src/app/globals.css` — base styles, glass effects, and animations

## Content Covered

- Professional summary and education
- Experience: NEXIUM AI Developer Internship + FAST TA role
- Featured projects with CV-aligned descriptions and GitHub links
- Skills marquee and AI/Data strengths
- Contact section and social links

## Production Checks

```bash
npm run lint
npm run build
```
