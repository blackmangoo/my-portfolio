# AI/ML Engineering Portfolio

Welcome to my portfolio repository. This project is built using modern full-stack technologies to showcase my projects, experience, and skills in AI/ML engineering.

## Project Structure

The Next.js 16 application is located in the `ammar-portfolio/` directory.

To run the application:

```bash
cd ammar-portfolio
npm install
npm run dev
```

The application is built with:
- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- TypeScript
